# Repository Agent Instructions

1. **문서 참조 필수**: 코드 또는 문서를 수정하기 전에 `docs/` 및 `MyDocuments/` 폴더의 관련 문서를 확인하고 요구사항과 일치하도록 반영하세요.
2. **언어 선택**: 백엔드 구현은 기본적으로 Python 3.11 + FastAPI 스택을 사용합니다. 다른 언어를 제안하거나 사용할 경우 제품 담당자의 승인을 문서화해야 합니다.
3. **보안 준수**: 개인정보 및 쿠폰번호는 암호화/마스킹 정책(`docs/data-model.md`, `docs/functional-spec.md`)을 반드시 따릅니다. 예제나 로그에 실제 값을 노출하지 마세요.
4. **문서 업데이트**: 요구사항이나 설계가 변경되면 관련 문서를 먼저 업데이트하고, 커밋 메시지에 `[docs]` 태그를 포함하세요.
5. **테스트 보고**: 코드 변경 시 실행한 테스트 명령과 결과를 PR/최종 응답에 명시하세요.
6. **AGENTS 확장**: 디렉터리별 세부 지침이 필요한 경우 해당 디렉터리에 추가 `AGENTS.md`를 생성하여 보강할 수 있습니다.
